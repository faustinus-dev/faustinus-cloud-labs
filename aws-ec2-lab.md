# AWS EC2 Deployment Lab

## Objective
Deploy and configure a secure EC2 instance and document the process.

## Steps Performed
- Created EC2 instance using Amazon Linux
- Configured security groups (SSH only)
- Generated key pair
- Connected via SSH
- Installed updates
- Installed NGINX web server
- Verified web service via public IP

## Security Measures
- Restricted inbound ports
- Used key-based authentication
- Disabled password login

## Outcome
Successfully deployed and secured a working EC2 server.

## Skills Demonstrated
AWS EC2
Security Groups
Linux Administration
SSH
Basic Server Hardening
